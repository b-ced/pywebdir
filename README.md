# PyWebDir
Create a web directory structure using a python script

This a small python script to quicly create a new web directory structure


## It will create the following structure :


        Dir
         │
         ├─ css
         │  │
         │  └─ style.css
         │
         ├─ js
         │  │
         │  └─ script.js
         │
         └─ index.php
